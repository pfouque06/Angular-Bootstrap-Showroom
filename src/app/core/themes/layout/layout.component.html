Customizing the grid

The number of grid columns can be modified via Sass variables

$grid-columns is used to generate the widths (in percent) of each individual colum while $grid-gutter-width sets the width for the column gutters : 

        $grid-columns: 12 !default;
        $grid-gutter-width: 30px !default;
