Usage

Prerequisites

Before proceeding to Fontawesome usage, it must be already installed in yout angular project has been created.

Please check "Foundations" HowTo to do so.

Icon selector :

Use the fa-icon semantic with [icon] directive to select icons.

     

Color :

Sizing directive :

Use the size attribute to define the size of the icone which is relative to its parent

<fa-icon [icon]='camera' size='[xs,sm,lg,2x,..,10x]'></fa-icon> :

Options

Fixed width directive :

Use the fixedWidth attribute to vertically assign a series of icons, like in a list or a navigation menu.

Listed icons directives :

Use the fa-ul and fa-li class directive to replace default bullets in unordered lists; fixedWidth attribute is handled as well.

  • List item checked
  • List item unchecked
  • List icons checked

Rotate & Flip directives :

Use the rotate and flip directive according to your needs.



Rotated animation directives :

Use the spin (linear) and pulse (punctuated) rotation directives according to your needs.



border, Left & right pulling directives :

Use pull='left' or pull='right' class directives in order to pull icons abd wrap text around if needed.

Use [border]='true' directive to add a light border around your icon.

Enhancement

Icon stacking directives :

Use the fa-stack semantic with stackItemSize='1x|2x'(order is important) and inverse directives as needed.


Power transforms directives :

You can scale, position, flip, & rotate icons arbitrarily using the data-fa-transform element attribute. You can even combine them for some super-useful effects.

Sub directives are :

  • shrink-# grow-#
  • up-# right-# down-# left-#
  • rotate-# flip-v flip-h
